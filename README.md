# My First C++ Project - Library Management System
This is my first project uploaded to GitHub! It's a complete library system written in C++.


## What This Program Does
- **Manage Books**: Add, edit, delete, and search for books
- **Customer System**: Register and login as a customer
- **Loan Books**: Customers can borrow books for 7 days
- **Fine System**: Automatic fine calculation for overdue books
  - 1 day late: RM 5
  - 2 days late: RM 6  
  - 3 days late: RM 7
  - 4+ days late: RM 10 per day
- **Payment Tracking**: Track fines and partial payments
